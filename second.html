Question-2: What is the difference between innerHTML, innerText and textContent?

When I call innerHTML, the browser console will show the full HTML of the element I declared innerText shows only the visible text, and textContent returns all the text content inside the element.

Question-3: What is event delegation in the DOM?

Event delegation is a technique there is a single event listener added to a parent element instead of multiple listeners.

Question-4: What is Event bubbling in the DOM?

Event bubbling is a process there is an event starts from the target element in the DOM tree, triggering event listeners on each element.

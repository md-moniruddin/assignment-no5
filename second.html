Question-2: What is the difference between innerHTML, innerText and textContent?

When I call innerHTML, the browser console will show the full HTML of the element I declared. innerText shows only the visible text, and textContent returns all the text content inside the element, including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element instead of multiple listeners to each child element. The event bubbles up, and the listener can handle events for child elements.

Question-4: What is Event bubbling in the DOM?

Event bubbling is a process where an event starts from the target element and bubbles up through its ancestors in the DOM tree, triggering event listeners on each ancestor element.
